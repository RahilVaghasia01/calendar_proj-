# WIZZ Space Calendar

## Overview
WIZZ Space Calendar is a frontend-only calendar web app that allows users to create an account, log in, and manage personal events. Users can add, edit, delete, and export events, with data stored locally in the browser using localStorage.

## Features
- User registration and login
- Monthly calendar view
- Add events to specific dates
- Edit existing events
- Delete events
- View upcoming events
- Export events as CSV
- Local browser storage for saved data

## Project Files
- `index.html` — main structure of the app
- `style.css` — styling and layout
- `app.js` — functionality and event logic
- `README.md` — project documentation

## How to Run
1. Download or clone the repository.
2. Make sure the following files are in the same folder:
   - `index.html`
   - `app.js`
   - `style.css`
3. Open `index.html` in a web browser.
